Introducción a JavaScript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

Dato importante: JavaScript no es lo mismo que Java. Son lenguajes diferentes con propósitos y características distintas.

¿Dónde se ejecuta JavaScript?

JavaScript se puede ejecutar en:

  • Navegadores web (Chrome, Firefox, Safari, Edge)
  • Servidores con Node.js
  • Dispositivos móviles
  • Bases de datos como MongoDB

Sintaxis básica de JavaScript

La sintaxis de JavaScript es similar a otros lenguajes de programación como C, Java o C#.

// Esto es un comentario de una línea

/*
 Esto es un comentario
 de múltiples líneas
*/

// Declaración de variable
let mensaje = "Hola Mundo";

// Función simple
function saludar() {
    return "¡Hola!";
}

// Mostrar resultado en consola
console.log(mensaje);
console.log(saludar());

Variables y tipos de datos

En JavaScript, tenemos tres formas de declarar variables: var, let y const.

// Variables con var (menos usada en código moderno)
var nombre = "Juan";

// Variables con let (valor puede cambiar)
let edad = 25;
edad = 26; // Esto es válido

// Variables con const (valor constante)
const PI = 3.1416;
// PI = 3.15; // Esto daría error

// Tipos de datos primitivos
let texto = "Hola"; // string
let numero = 42; // number
let esMayor = true; // boolean
let nada = null; // null
let noDefinido; // undefined
let simbolo = Symbol('id'); // symbol

Recomendación: Usa const por defecto y solo let cuando necesites reasignar valores. Evita var en código moderno.

Operadores en JavaScript

JavaScript tiene operadores aritméticos, de comparación, lógicos y de asignación.

// Operadores aritméticos
let suma = 5 + 3; // 8
let resta = 10 - 4; // 6
let multiplicacion = 3 * 4; // 12
let division = 20 / 5; // 4
let modulo = 15 % 4; // 3 (resto de la división)

// Operadores de comparación
let igual = 5 == '5'; // true (igualdad débil)
let estrictamenteIgual = 5 === '5'; // false (igualdad estricta)
let diferente = 5 != '5'; // false
let estrictamenteDiferente = 5 !== '5'; // true

// Operadores lógicos
let and = (5 > 3) && (2 < 4); // true
let or = (5 > 7) || (3 < 5); // true
let not = !(5 > 3); // false

Estructuras de control

Las estructuras de control permiten dirigir el flujo de ejecución del programa.

Condicionales

// if...else
let edad = 18;

if (edad >= 18) {
    console.log("Eres mayor de edad");
} else {
    console.log("Eres menor de edad");
}

// switch
let dia = "Lunes";

switch (dia) {
    case "Lunes":
        console.log("Inicio de semana");
        break;
    case "Viernes":
        console.log("¡Fin de semana cerca!");
        break;
    default:
        console.log("Día normal");
}

Bucles

// for
for (let i = 0; i < 5; i++) {
    console.log("Iteración número: " + i);
}

// while
let contador = 0;
while (contador < 3) {
    console.log("Contador: " + contador);
    contador++;
}

// do...while
let x = 0;
do {
    console.log("Valor de x: " + x);
    x++;
} while (x < 3);

Funciones en JavaScript

Las funciones son bloques de código reutilizables que realizan una tarea específica.

// Declaración de función
function sumar(a, b) {
    return a + b;
}

// Expresión de función
const multiplicar = function(a, b) {
    return a * b;
};

// Función flecha (arrow function) - ES6
const dividir = (a, b) => {
    return a / b;
};

// Función flecha con retorno implícito
const cuadrado = n => n * n;

// Llamando a las funciones
console.log(sumar(5, 3)); // 8
console.log(multiplicar(4, 5)); // 20
console.log(dividir(10, 2)); // 5
console.log(cuadrado(6)); // 36

Objetos y arrays

Los objetos y arrays son estructuras de datos fundamentales en JavaScript.

Arrays

// Creación de arrays
let frutas = ["manzana", "banana", "naranja"];
let numeros = [1, 2, 3, 4, 5];

// Acceder a elementos
console.log(frutas[0]); // "manzana"
console.log(numeros[2]); // 3

// Métodos comunes de arrays
frutas.push("uva"); // Añade al final
frutas.pop(); // Elimina el último elemento
frutas.unshift("fresa"); // Añade al inicio
let longitud = frutas.length; // Longitud del array

Objetos

// Creación de objetos
let persona = {
    nombre: "Ana",
    edad: 30,
    ciudad: "Madrid",
    saludar: function() {
        return "Hola, soy " + this.nombre;
    }
};

// Acceder a propiedades
console.log(persona.nombre); // "Ana"
console.log(persona["edad"]); // 30
