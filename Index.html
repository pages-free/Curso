Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Describe la estructura de una página web mediante etiquetas (tags).

Tip: HTML no es un lenguaje de programación, es un lenguaje de marcado que define la estructura de tu contenido.

Breve historia de HTML

HTML fue creado por Tim Berners-Lee en 1991. La primera versión pública fue HTML 2.0 en 1995. HTML 4.01 se publicó en 1999 y fue una versión muy importante. HTML5, la versión actual, se publicó en 2014 y introdujo muchas características nuevas para soportar multimedia y aplicaciones web modernas.

Versiones de HTML

Versión Año Características principales
HTML 2.0 1995 Primera versión estandarizada
HTML 3.2 1997 Soporte para tablas y applets
HTML 4.01 1999 Hojas de estilo, scripting, marcos
XHTML 1.0 2000 HTML reformulado como XML
HTML5 2014 Elementos semánticos, multimedia, APIs

Estructura básica de un documento HTML

Todo documento HTML sigue una estructura básica que incluye la declaración de tipo, html, head y body.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi primera página</title>
    <!-- Enlaces a hojas de estilo -->
    <link rel="stylesheet" href="styles.css">
    <!-- Scripts -->
    <script src="script.js" defer></script>
</head>
<body>
    <!-- Contenido de la página -->
    <h1>¡Hola Mundo!</h1>
    <p>Este es mi primer párrafo.</p>
</body>
</html>

Importante: La declaración <!DOCTYPE html> debe ser lo primero en un documento HTML. Le indica al navegador que se trata de un documento HTML5.

Partes de un elemento HTML

Un elemento HTML generalmente consta de:

  • Etiqueta de apertura: <nombre>
  • Contenido: El contenido del elemento
  • Etiqueta de cierre: </nombre>

Algunos elementos, como <img>, son elementos vacíos y no tienen contenido ni etiqueta de cierre.
